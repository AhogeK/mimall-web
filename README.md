# mimall-web
小米商城前端学习项目

## 项目结构

```
.
|- public         公共文件
|  |- imgs        图片
|  |- mock        接口信息
|- resurce        相关文档
|- src            源码
|  |- api         接口调用相关
|  |- assets      经过webpack处理的资源
|  |- components  组件
|  |- pages       页面
|  |- storage     本地存储
|  |- util        工具
.
```
