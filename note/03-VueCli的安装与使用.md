### VueCli 4.0 安装

具体详细可以访问官方文档[安装一栏](https://cli.vuejs.org/zh/guide/installation.html)

* ``npm install -g @vue/cli``
* ``yarn global add @vue/cli``
* ``vue --version``

### VueCli 4.0 使用

``vue create mall`` 创建项目

创建时会有提示预安装哪些插件 可以选择默认

``vue ui`` 可以启动Vue项目的管理页面

进入管理页面可以进行一些必要插件的安装
* axios
* vue-router
* vuex
